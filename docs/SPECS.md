## 1. API Contract

### 1.1 Transport & Conventions

| Aspect | Specification |
| --- | --- |
| Protocol | REST (Spring MVC + SpringDoc OpenAPI 3) |
| Base Prefix | `/api/v1` |
| Gateway Routing | Via `ai-gateway-core` (§4.1.2) |
| Internal gRPC | `ControlPlaneClient` for `ai-admin-service` ↔ `ai-platform-api` (proto in `specs/`) |
| Auth Header | `Authorization: Bearer <JWT>` + `X-Tenant-Id` |
| Response Envelope | `ApiResponse<T>` wrapping all responses |
| OpenAPI | `openapi.yaml` auto-generated by SpringDoc, archived in `specs/` |

### 1.3 HTTP Status Codes

| Code | Meaning |
| --- | --- |
| `2xx` | Success |
| `400` | Parameter / business rule violation |
| `401` | Unauthenticated |
| `403` | Unauthorized / model not authorized |
| `404` | Aggregate root not found |
| `409` | `tenant_id` conflict / quota conflict |
| `422` | Manifest dependency validation failure (§12.4) |
| `429` | Tenant QPS quota exhausted |
| `500` | Internal server error |

### 1.4 Error Response Schema

```json
{
  "code": "TENANT_QUOTA_GPU_DISABLED",
  "message": "GPU quota only effective when self-hosted inference (full profile modelServing) is enabled",
  "traceId": "b3b1...",
  "doc": "https://docs.openstrata.cc/errors/TENANT_QUOTA_GPU_DISABLED"
}
```

### 1.5 Business Error Codes

| Code | HTTP | Description |
| --- | --- | --- |
| `TENANT_ID_CONFLICT` | 409 | `tenant_id` already exists |
| `TENANT_NOT_FOUND` | 404 | Tenant aggregate does not exist |
| `TENANT_QUOTA_GPU_DISABLED` | 400 | GPU quota only for full profile self-hosted inference |
| `MULTITENANCY_REQUIRES_AUTH` | 422 | `auth` must be enabled before `multitenancy` |
| `ENTITLEMENT_DEP_VIOLATION` | 422 | Component whitelist violates `PlatformManifest` dependency graph |
| `MODEL_RESTRICTED` | 403 | Model requires Enterprise plan |
| `APPROVAL_REQUIRED` | 403 | High-risk operation requires approval |
| `QUOTA_CONFLICT` | 409 | Quota dimension conflict |

## 2. Data Model

### 2.1 Database

| Attribute | Value |
| --- | --- |
| Engine | PostgreSQL 16.0 (core, §16 base) |
| Schema | `platform_cp` (shared) |
| Multi-tenant isolation | `tenant_id` column + RLS; Schema-per-tenant + RLS after multi-tenancy lit |
| ORM | JPA (Hibernate 6 / Jakarta Persistence) |
| Migration | Flyway (`V1__init.sql`, `V2__rls.sql`) |

### 2.2 Table DDL

```sql
-- Tenants
CREATE TABLE tenants (
  tenant_id     VARCHAR(64) PRIMARY KEY,
  name          VARCHAR(128) NOT NULL,
  status        VARCHAR(16)  NOT NULL DEFAULT 'PROVISIONING',
  plan_id       VARCHAR(32),
  multitenancy  BOOLEAN      NOT NULL DEFAULT FALSE,
  created_at    TIMESTAMPTZ  NOT NULL DEFAULT now(),
  updated_at    TIMESTAMPTZ  NOT NULL DEFAULT now()
);

-- Users
CREATE TABLE users (
  user_id    VARCHAR(64) PRIMARY KEY,
  tenant_id  VARCHAR(64) NOT NULL REFERENCES tenants(tenant_id),
  email      VARCHAR(256) NOT NULL,
  role       VARCHAR(16)  NOT NULL,
  status     VARCHAR(16)  NOT NULL DEFAULT 'INVITED',
  kc_id      VARCHAR(64),
  UNIQUE (tenant_id, email)
);

-- Applications
CREATE TABLE applications (
  app_id     VARCHAR(64) PRIMARY KEY,
  tenant_id  VARCHAR(64) NOT NULL REFERENCES tenants(tenant_id),
  name       VARCHAR(128) NOT NULL,
  agentspec_ref VARCHAR(256)
);

-- Plans
CREATE TABLE plans (
  plan_id         VARCHAR(32) PRIMARY KEY,
  name            VARCHAR(64) NOT NULL,
  quota_template  JSONB NOT NULL
);

-- Quotas
CREATE TABLE quotas (
  quota_id   VARCHAR(64) PRIMARY KEY,
  tenant_id  VARCHAR(64) NOT NULL REFERENCES tenants(tenant_id),
  dimension  VARCHAR(16) NOT NULL,
  limit_val  BIGINT NOT NULL,
  used_val   BIGINT NOT NULL DEFAULT 0,
  UNIQUE (tenant_id, dimension)
);

-- Entitlements
CREATE TABLE entitlements (
  tenant_id  VARCHAR(64) NOT NULL REFERENCES tenants(tenant_id),
  component  VARCHAR(48) NOT NULL,
  allowed    BOOLEAN NOT NULL DEFAULT FALSE,
  PRIMARY KEY (tenant_id, component)
);

-- Model Grants
CREATE TABLE model_grants (
  tenant_id VARCHAR(64) NOT NULL REFERENCES tenants(tenant_id),
  provider  VARCHAR(32) NOT NULL,
  model     VARCHAR(64) NOT NULL,
  PRIMARY KEY (tenant_id, provider, model)
);

-- Audit Log (immutable)
CREATE TABLE audit_log (
  id          BIGSERIAL PRIMARY KEY,
  tenant_id   VARCHAR(64),
  actor       VARCHAR(64) NOT NULL,
  action      VARCHAR(64) NOT NULL,
  payload     JSONB,
  created_at  TIMESTAMPTZ NOT NULL DEFAULT now()
);
```

## 3. Configuration

### 3.1 Service Configuration (`application.yml` / `infrastructure/config/platform-api.yaml`)

```yaml
openstrata:
  service:
    port: 8081
  features:
    multitenancy:
      enabled: false              # starter/standard=false; advanced/full=true
    billing:
      enabled: false              # only multi-tenant (advanced/full)
    security:
      enabled: false              # advanced guardrails via ai-srs-service
    riskControl:
      enabled: true               # core always on: rate-limit + injection/PII scan
  spi:
    auth:
      provider: keycloak
    multitenancy:
      provider: capsule          # optional, only when multi-tenancy lit
    cache:
      provider: redis             # default; valkey as OSI alternative
```

### 3.2 Profile → Feature Matrix

| Profile | multitenancy | billing | security | riskControl | Cache | MultiTenancy Provider |
| --- | --- | --- | --- | --- | --- | --- |
| starter | false | false | false | true | redis | — (null adapter) |
| standard | false | false | false | true | redis | — (null adapter) |
| advanced | true | true | false | true | redis | capsule |
| full | true | true | true | true | redis | capsule |

### 3.3 Configuration Keys Reference

| Key | Type | Default | Profiles | Description |
| --- | --- | --- | --- | --- |
| `openstrata.service.port` | int | `8081` | all | HTTP listen port |
| `openstrata.features.multitenancy.enabled` | bool | `false` | all | Enable multi-tenancy |
| `openstrata.features.billing.enabled` | bool | `false` | advanced/full | Enable billing linkage |
| `openstrata.features.security.enabled` | bool | `false` | full | Enable advanced guardrails (ai-srs-service) |
| `openstrata.features.riskControl.enabled` | bool | `true` | all | Core risk control (rate-limit, injection/PII scan) |
